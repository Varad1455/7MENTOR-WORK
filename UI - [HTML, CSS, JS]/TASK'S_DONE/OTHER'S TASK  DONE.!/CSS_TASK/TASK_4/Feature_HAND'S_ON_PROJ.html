What You Will Build

Our curriculum is built around practical application. From day one, you'll be coding and building projects that are designed to give you the experience employers are looking for. Here are some of the key projects you'll complete:

Project 1: Personal Portfolio Website

Create a stunning, responsive personal portfolio from scratch using HTML and CSS. Learn to structure content, apply modern styling, and deploy it live on the web.

  • Skills Gained: Semantic HTML, CSS Flexbox & Grid, Responsive Design.
  • Outcome: A professional online presence to showcase your future work.

Project 2: Interactive To-Do List App

Dive into JavaScript by building a dynamic to-do list application. Implement features like adding, deleting, and marking tasks as complete, all saved in the browser's local storage.

  • Skills Gained: JavaScript DOM Manipulation, Event Handling, Local Storage.
  • Outcome: A functional web app that demonstrates your front-end logic skills.

Project 3: Weather Dashboard with API

Connect to a third-party API to fetch and display real-time weather data. This project teaches you how to handle asynchronous JavaScript, work with JSON data, and present it in a user-friendly interface.

  • Skills Gained: Asynchronous JavaScript (Fetch API), JSON, Error Handling.
  • Outcome: A dynamic dashboard that interacts with external services.
