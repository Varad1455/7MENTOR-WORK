Fixed-Width Container

This section uses the .container class. It provides a responsive fixed-width container, which means its max-width changes at each breakpoint. It's centered on the page and is great for standard content layouts.

Full-Width Container

This section uses the .container-fluid class, which is always 100% wide, spanning the entire width of the viewport. It's perfect for full-width hero sections, banners, or footers. I've added a dark background and some padding to make it stand out.

Notice how this content is back within the fixed-width layout, demonstrating the clear difference between the two container types.
